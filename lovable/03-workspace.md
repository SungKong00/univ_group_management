# 기능 명세: 워크스페이스

## 1. 목표
사용자는 자신이 속한 그룹의 전용 협업 공간(워크스페이스)에 접근하여, 공지사항을 확인하고, 채널별로 소통하며, 소속된 멤버 정보를 파악할 수 있다.

## 2. 화면 구성 및 기능 (기능/UI·UX 명세 기반)

### 2.1. 진입점
- **홈 화면:** 로그인 후 홈 화면에서 사용자가 속한 워크스페이스 목록이 카드 형태로 표시된다.
- **워크스페이스 카드:** 각 카드는 그룹의 이름, 설명, 멤버 수 등 요약 정보를 보여준다.
- **진입:** 워크스페이스 카드를 클릭하면 해당 그룹의 워크스페이스 메인 화면으로 진입한다.

### 2.2. 워크스페이스 메인 화면
- **화면 구조:** 상단바와 탭 기반의 콘텐츠 영역으로 구성된다.
    - **상단바:** 현재 워크스페이스의 이름, 사용자의 역할 배지, (권한자일 경우) 관리 버튼이 표시된다.
    - **탭 네비게이션:** '공지', '채널', '멤버'의 3가지 핵심 기능으로 전환할 수 있는 탭이 제공된다.

### 2.3. 핵심 기능 (탭별)

#### 2.3.1. 공지 탭
- **목표:** 그룹의 중요 공지사항을 확인한다.
- **기능:**
    - 그룹장 또는 관리 권한이 있는 멤버가 작성한 공지사항들이 카드 리스트 형태로 최신순으로 표시된다.
    - 중요한 공지는 상단에 고정(pinned)될 수 있다.
    - 권한이 있는 사용자는 '공지 작성' 버튼을 통해 새 공지를 작성할 수 있다.

#### 2.3.2. 채널 탭
- **목표:** 주제별 채널에 참여하여 멤버들과 실시간으로 소통한다.
- **기능:**
    - **채널 목록:** '자유', 'Q&A' 등 워크스페이스 내에 생성된 채널 목록이 표시된다.
    - **채널 상세 (채팅 피드):**
        - 특정 채널을 선택하면, 시간 순서대로 정렬된 메시지 피드가 나타난다.
        - 각 메시지는 작성자 프로필, 닉네임, 본문, 시간, 리액션, 댓글 수를 포함한다.
        - 화면 하단에는 메시지와 파일을 전송할 수 있는 입력창(Composer)이 고정되어 있다.
    - **권한:** 채널별로 설정된 열람/작성 권한에 따라 보기 및 메시지 작성이 제한될 수 있다.

#### 2.3.3. 멤버 탭
- **목표:** 워크스페이스에 소속된 전체 멤버를 확인하고 관리한다.
- **기능:**
    - **멤버 목록:** 소속된 멤버의 총 인원수와 함께, 각 멤버의 프로필 사진, 닉네임, 역할이 리스트 형태로 표시된다.
    - **검색 및 필터:** 멤버의 이름으로 검색하거나 역할(예: '스태프', '일반 멤버')별로 필터링하여 볼 수 있다.
    - **관리 기능 (권한자):**
        - 관리 권한이 있는 사용자는 각 멤버 항목의 '더보기(...)' 메뉴를 통해 역할을 변경하거나 멤버를 그룹에서 제외(강제 탈퇴)시킬 수 있다.
        - 그룹장은 자신의 '더보기' 메뉴를 통해 다른 멤버에게 그룹장 권한을 위임할 수 있다.

### 2.4. 신규 워크스페이스 생성
- **위치:** 홈 화면(워크스페이스 목록)에 '새 워크스페이스 생성' 버튼이 제공된다.
- **동작:**
    - 버튼 클릭 시, 워크스페이스의 이름과 설명을 입력하는 폼이 포함된 모달(Modal) 창이 나타난다.
    - 정보 입력 후 제출하면, 별도의 승인 절차 없이 즉시 독립 그룹과 워크스페이스가 생성된다.
    - 생성 완료 후, 사용자의 워크스페이스 목록이 자동으로 새로고침되어 방금 만든 워크스페이스가 표시된다.
