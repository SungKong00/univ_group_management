## 🔧 기술적 접근

### 1. 자동화 스크립트
- **check-broken-links.sh**: 깨진 링크 자동 검증
  - 정규 표현식 기반 링크 추출
  - 상대 경로 해석 및 파일 존재 확인
  - CSV 및 마크다운 리포트 생성

- **update-links.sh** (예정): 링크 자동 수정
  - sed 기반 일괄 치환
  - 백업 생성 후 실행
  - 수정 후 검증 재실행

### 2. 문서 분할 전략
- **원칙**: 논리적 단위 기준 (기능별, Phase별, 도메인별)
- **구조**: 인덱스 파일 + 하위 상세 파일
- **깊이**: 최대 2단계 계층
- **링크**: 분할 전 추출 → 분할 후 자동 업데이트

### 3. 컨텍스트 추적
- **sync-status.md**: 전체 문서 동기화 상태
- **pending-updates.md**: 업데이트 대기 목록
- **context-update-log.md**: 변경 이력
- **자동화**: 커밋 메시지 기반 영향 문서 추출
