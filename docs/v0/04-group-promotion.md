# React V0 개발 프롬프트: 4단계 - 그룹 홍보 및 모집 기능

## 1. 목표
서비스에 가입하지 않은 사용자도 볼 수 있는 공개적인 그룹 홍보 페이지를 만들고, 사용자가 그룹에 가입 신청을 할 수 있는 기능을 구현합니다.

## 2. 참고 명세
-   **기능 명세:** `docs/설계 문서/기능명세서/4 홍보 모집.md`
-   **UI/UX 명세:** `docs/설계 문서/UI:UX 명세서/4. 홍보_모집_uiux.md`

## 3. 상세 구현 단계

### 3.1. 그룹 홍보 목록 페이지
1.  **페이지 생성 (`src/pages/promotion/PromotionListPage.tsx`)**
    -   누구나 접근할 수 있는 공개 페이지입니다.
    -   백엔드 API (`/api/promotions`)를 호출하여 현재 모집 중인 그룹 홍보 게시글 전체 목록을 가져옵니다.
    -   MUI `Grid`와 `Card`를 사용하여 각 홍보 게시글을 요약하여 보여줍니다. (그룹명, 간단한 소개, 모집 분야 등)
    -   카드 클릭 시 홍보 상세 페이지로 이동합니다.

2.  **검색 및 필터링 기능**
    -   페이지 상단에 검색창(그룹명, 소개 내용 등)과 필터(모집 분야, 활동 지역 등) 옵션을 배치합니다.
    -   검색/필터 조건이 변경될 때마다 API를 다시 호출하여 목록을 갱신합니다.

### 3.2. 그룹 홍보 상세 페이지
1.  **페이지 생성 (`src/pages/promotion/PromotionDetailPage.tsx`)**
    -   URL 파라미터(`promotionId`)를 기반으로 특정 홍보 게시글의 상세 정보를 API로 조회하여 화면에 표시합니다.
    -   그룹에 대한 자세한 소개, 활동 내용, 모집 요강 등을 보여줍니다.
    -   페이지 하단 또는 측면에 '가입 신청하기' 버튼을 배치합니다.

### 3.3. 가입 신청 기능
1.  **가입 신청 모달 (`src/components/promotion/ApplyModal.tsx`)**
    -   '가입 신청하기' 버튼 클릭 시 나타나는 모달입니다.
    -   (로그인하지 않은 경우) 로그인 페이지로 이동하라는 안내와 링크를 보여줍니다.
    -   (로그인한 경우) 가입 신청서(간단한 자기소개, 지원 동기 등)를 작성할 수 있는 폼을 제공합니다.

2.  **API 연동**
    -   폼 제출 시, 가입 신청 API (`/api/promotions/{promotionId}/apply`)를 호출합니다.
    -   성공적으로 제출되면 "가입 신청이 완료되었습니다." 라는 메시지를 보여주고 모달을 닫습니다.

## 4. 완료 조건
-   로그인하지 않아도 그룹 홍보 목록 페이지와 상세 페이지에 접근할 수 있다.
-   홍보 목록 페이지에서 검색 및 필터 기능이 동작한다.
-   홍보 상세 페이지에서 '가입 신청하기' 버튼을 누르면, 로그인 여부에 따라 적절한 화면(로그인 안내 또는 신청 폼)이 나타난다.
-   로그인한 사용자는 가입 신청서를 작성하고 제출할 수 있다.
