# React V0 개발 프롬프트: 3단계 - 워크스페이스 내부 기능 (게시글 및 멤버 관리)

## 1. 목표
사용자가 특정 워크스페이스에 진입하여 게시글을 작성 및 조회하고, 해당 워크스페이스의 멤버를 관리하는 기능을 구현합니다.

## 2. 선행 조건
- 2단계에서 구현된 워크스페이스 목록 조회가 가능해야 합니다.
- 워크스페이스 목록에서 특정 워크스페이스를 클릭했을 때, 해당 워크스페이스의 `id`를 파라미터로 하여 이 페이지로 진입할 수 있어야 합니다.

## 4. 상세 구현 단계

### 4.1. 워크스페이스 상세 페이지 레이아웃
1.  **페이지 생성 (`src/pages/workspace/WorkspaceDetailPage.tsx`)**
    -   React Router의 `useParams`를 사용하여 URL로부터 `workspaceId`를 가져옵니다.
    -   MUI의 `Tabs` 컴포넌트를 사용하여 '게시판', '멤버 관리' 등 워크스페이스 내부 기능을 전환할 수 있는 UI를 구현합니다.
    -   페이지 상단에는 현재 워크스페이스의 이름과 정보를 표시합니다.

### 4.2. 게시판 기능 구현
1.  **게시글 목록 컴포넌트 (`src/components/post/PostList.tsx`)**
    -   '게시판' 탭 선택 시 이 컴포넌트를 렌더링합니다.
    -   `workspaceId`를 기반으로 백엔드 API(`/api/workspaces/{workspaceId}/posts`)를 호출하여 게시글 목록을 가져옵니다.
    -   MUI의 `Table` 또는 `List` 컴포넌트를 사용하여 게시글 제목, 작성자, 작성일 등을 표시합니다.
    -   '글쓰기' 버튼을 포함하여, 클릭 시 게시글 작성 페이지로 이동합니다.
    -   목록의 각 항목 클릭 시 게시글 상세 페이지로 이동합니다.

2.  **게시글 작성/수정 페이지 (`src/pages/post/PostEditorPage.tsx`)**
    -   MUI `TextField`를 사용하여 제목과 내용(Multiline)을 입력받는 폼을 구현합니다.
    -   '저장' 버튼 클릭 시, 생성 또는 수정 API를 호출합니다.
    -   (수정 모드) 기존 게시글의 `postId`가 URL 파라미터로 전달되면, 기존 데이터를 불러와 폼에 채워줍니다.

3.  **게시글 상세 보기 페이지 (`src/pages/post/PostDetailPage.tsx`)**
    -   게시글의 제목, 내용, 작성자, 작성일 등을 표시합니다.
    -   (본인 게시글일 경우) '수정', '삭제' 버튼을 제공합니다.
    -   **댓글 기능 (CRUD):**
        -   댓글 목록을 조회하여 보여줍니다.
        -   새 댓글을 작성하고 등록할 수 있는 입력 폼을 구현합니다.
        -   (본인 댓글일 경우) 수정, 삭제 기능을 제공합니다.

### 4.3. 멤버 관리 기능 구현
1.  **멤버 목록 컴포넌트 (`src/components/member/MemberList.tsx`)**
    -   '멤버 관리' 탭 선택 시 이 컴포넌트를 렌더링합니다.
    -   백엔드 API(`/api/workspaces/{workspaceId}/members`)를 호출하여 멤버 목록을 가져옵니다.
    -   MUI `Table`을 사용하여 각 멤버의 닉네임, 역할(Role) 등을 표시합니다.
    -   (관리자 권한일 경우) 멤버의 역할을 변경할 수 있는 드롭다운 메뉴와 '멤버 초대' 버튼을 구현합니다.

2.  **멤버 초대 모달 (`src/components/member/InviteMemberModal.tsx`)**
    -   '멤버 초대' 버튼 클릭 시 나타나는 모달입니다.
    -   초대할 사용자의 이메일 또는 닉네임을 입력받는 필드를 구현합니다.
    -   초대 API를 호출하여 멤버를 초대합니다.

## 5. 완료 조건
-   워크스페이스 상세 페이지에 접속하면 '게시판'과 '멤버 관리' 탭이 보인다.
-   '게시판' 탭에서 해당 워크스페이스의 게시글 목록이 나타나고, 글쓰기 및 상세 보기로 이동이 가능하다.
-   게시글 상세 보기 페이지에서 댓글을 작성, 수정, 삭제할 수 있다.
-   '멤버 관리' 탭에서 멤버 목록과 역할을 볼 수 있다.
-   (권한이 있는 경우) 멤버의 역할을 변경하거나 새로운 멤버를 초대할 수 있다.
