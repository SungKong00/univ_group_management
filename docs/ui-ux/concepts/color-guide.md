# Web Application Color System Guide

**[우선순위] 라이트 모드(Light Mode)를 먼저 구현합니다.**

이 문서는 웹 애플리케이션의 일관된 사용자 경험을 위한 컬러 시스템을 정의합니다. 모든 UI 컴포넌트 개발 시 본 가이드를 참고해 주시기 바랍니다.

---

## 🎨 라이트 모드 (Light Mode)

### 1. 컨셉

> **창의성, 활기, 동기 부여.** 밝고 경쾌한 시각적 경험을 제공하여 사용자의 적극적인 참여를 유도합니다.

### 2. 색상 팔레트 및 역할 정의

| 색상 | Hex 코드 | 역할 (라이트 모드) | 상세 사용 가이드 |
| :--- | :--- | :--- | :--- |
| **메인 퍼플** | `#5E239D` | **Brand Accent** | - 브랜드 로고, 헤더, 주요 일러스트 등 브랜드 정체성을 드러내는 곳에 사용합니다. <br> - 밝은 배경과 잘 어울리며, 브랜드의 고급감을 유지합니다. |
| **라이트 배경** | `#FFFFFF` | **Primary Surface** | - 화면의 **기본 배경색** (60%). 깨끗하고 밝은 느낌을 제공합니다. <br> - 카드, 모달 등 모든 기본 컴포넌트의 배경으로 사용합니다. |
| **기본 텍스트/UI** | `#121212` | **Primary Text & UI** | - **라이트 배경** 위에서 가장 중요한 **모든 텍스트** (본문, 헤드라인)에 사용합니다. <br> - 기본 아이콘, 입력 필드의 플레이스홀더 텍스트로 사용합니다. |
| **보조 텍스트/경계**| `#6C757D` | **Secondary Text & Border** | - 중요도가 낮은 보조 텍스트, 설명 글에 사용합니다. <br> - 카드, 패널 등 UI 요소의 옅은 테두리 또는 구분선으로 사용합니다. <br> - 비활성화된 버튼의 텍스트/아이콘으로 사용합니다. |
| **일렉트릭 블루**| `#007BFF` | **Primary Action** | - **가장 중요한 CTA 버튼** (예: '참여하기', '업로드')에 사용합니다. <br> - 링크 텍스트, 활성화된 탭 또는 메뉴에 적용합니다. |
| **비비드 민트**| `#00F5D4` | **Success / Positive** | - '성공적으로 완료되었습니다'와 같은 **긍정적인 피드백 메시지**에 사용합니다. <br> - 새로운 알림, 신규 콘텐츠를 나타내는 뱃지(Badge)로 사용합니다. |
| **에너제틱 코랄**| `#FF5A5F` | **Danger / Alert** | - **삭제, 탈퇴** 등 되돌리기 어려운 위험한 작업에 대한 확인 버튼에 사용합니다. <br> - 입력 오류, 시스템 경고 등 사용자 주의가 필요한 메시지에 적용합니다. |

### 3. 인터랙션 상태 (Interaction States)

-   **Hover (마우스 오버):**
    -   **일렉트릭 블루 버튼:** 채도를 유지하며 살짝 밝게 (`#3395FF`) 처리합니다.
    -   **리스트 아이템:** 배경색을 살짝 추가 (`rgba(0, 0, 0, 0.03)`)하여 선택될 영역을 표시합니다.
-   **Disabled (비활성화):**
    -   **모든 버튼 및 UI 요소:** 배경은 `#E9ECEF` (매우 옅은 그레이), 텍스트는 `#ADB5BD` (중간 그레이)로 처리하여 비활성화 상태를 명확히 보여줍니다.

### 4. 컴포넌트별 적용 예시

| 컴포넌트 | 스타일 (라이트 모드) |
| :--- | :--- |
| **주요 버튼** | `background-color: #007BFF;` <br> `color: #FFFFFF;` |
| **보조 버튼** | `background-color: transparent;` <br> `border: 1px solid #6C757D;` <br> `color: #6C757D;` |
| **알림 (성공)** | `background-color: rgba(0, 245, 212, 0.1);` <br> `border: 1px solid #00F5D4;` <br> `color: #008B7A;` (가독성을 위한 어두운 민트) |
| **알림 (오류)** | `background-color: rgba(255, 90, 95, 0.1);` <br> `border: 1px solid #FF5A5F;` <br> `color: #FF5A5F;` |
| **입력 필드** | `background-color: #FFFFFF;` <br> `border: 1px solid #CED4DA;` <br> `focus 시 border-color: #007BFF;` |
| **카드/패널** | `background-color: #FFFFFF;` <br> `border: 1px solid #E9ECEF;` 또는 `box-shadow` 사용 |

## 🎨 다크 모드 (Dark Mode)

### 1. 컨셉

> **창의성, 활기, 동기 부여.** 깊이감 있는 배경으로 콘텐츠의 몰입도를 높이고, 강조 색상을 통해 활기찬 브랜드 경험을 제공합니다.
