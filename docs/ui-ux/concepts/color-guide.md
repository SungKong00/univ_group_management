# 컬러 가이드 (Color Guide)

본 문서는 프로젝트의 일관된 사용자 경험을 위한 공식 컬러 시스템을 정의합니다.

## 1. 컬러 시스템 구조

컬러 시스템은 역할에 따라 네 가지 카테고리로 구성됩니다.

-   **Primary (주색):** 브랜드 정체성을 드러내는 핵심 색상
-   **Secondary (보조색):** Primary를 보완하고 가독성을 높이는 색상
-   **Neutral (중립색):** 텍스트, 배경, 구분선 등 UI의 기반을 이루는 색상
-   **Accent (강조색):** 사용자의 상호작용(버튼, 링크, 알림 등)에 포인트를 주는 색상

---

## 2. 컬러 팔레트 및 배치 가이드

### Primary (주색)

-   **Violet (보라):** `#6A1B9A`
    -   헤더/푸터 배경
    -   주요 버튼(CTA) 배경
    -   강조 텍스트(소제목, 중요한 키워드)
-   **Black (검정):** `#000000`
    -   로고, 메인 텍스트
    -   고대비가 필요한 UI 요소

### Secondary (보조색)

-   **Light Violet:** `#9C27B0`
    -   Hover 효과
    -   카드/박스 헤더 라인
    -   인포그래픽 색상 포인트
-   **Deep Violet:** `#4A148C`
    -   버튼 활성화(Active) 상태
    -   강조형 배너, 알림
    -   링크 Hover 색상

### Neutral (중립색)

-   **Gray Dark:** `#424242`
    -   본문 텍스트
    -   아이콘 및 서브 타이틀
-   **Gray Light:** `#E0E0E0`
    -   구분선(Border)
    -   섹션 배경
    -   카드 배경
-   **White:** `#FFFFFF`
    -   기본 배경
    -   텍스트 대비용

### Accent (강조색)

-   **Soft Lavender:** `#E1BEE7`
    -   배경 강조(하이라이트 섹션)
    -   버튼 보조색
    -   인포박스, 태그, 알림

---

## 3. 사용 비율 가이드

전체적인 디자인의 균형을 위해 아래의 비율을 권장합니다.

-   **Neutral (회색 계열 + 흰색):** 50%
-   **Primary (보라 + 검정):** 40%
-   **Accent (라벤더 계열):** 10%

👉 이 비율을 따르면 사이트가 전체적으로 깔끔하면서도 브랜드 정체성을 효과적으로 드러낼 수 있습니다.

---

## 4. UI 컴포넌트 적용 예시

-   **헤더(Header):** Violet (`#6A1B9A`) 배경 + White 텍스트
-   **메인 CTA 버튼:** Violet (`#6A1B9A`) 배경 + White 텍스트, Hover 시 Deep Violet (`#4A148C`)
-   **서브 버튼:** Gray Light (`#E0E0E0`) 배경 + Violet (`#6A1B9A`) 텍스트
-   **본문(Text):** Black (`#000000`) 또는 Gray Dark (`#424242`)
-   **섹션 배경:** White (`#FFFFFF`) 기본, 구분 필요시 Gray Light (`#E0E0E0`) / Soft Lavender (`#E1BEE7`)
-   **링크 색상:** Violet (`#6A1B9A`), Hover 시 Deep Violet (`#4A148C`)

---

## 5. 접근성 고려

모든 사용자가 콘텐츠를 명확하게 인지할 수 있도록 웹 접근성 가이드라인을 준수해야 합니다.

-   텍스트와 배경의 명도 대비(Contrast Ratio)는 **WCAG 2.1 AA 기준인 4.5:1 이상**을 확보해야 합니다.
-   **좋은 예시:** Violet (`#6A1B9A`) 배경과 White (`#FFFFFF`) 텍스트는 대비비율 7.8:1로 기준을 충족합니다.
-   **주의사항:** Gray Light (`#E0E0E0`) 배경 위에는 반드시 Black (`#000000`) 또는 Deep Violet (`#4A148C`) 같이 충분한 대비가 확보되는 텍스트 색상을 사용해야 합니다.
