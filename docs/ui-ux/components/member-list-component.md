# 멤버 필터 컴포넌트 (Member Filter Component)

멤버 목록을 필터링하고 표시하기 위한 UI 컴포넌트 설계입니다.

## 컴포넌트 계층 구조

```
MemberListView (최상위)
├── MemberFilterPanel (필터 영역)
│   ├── RoleFilter (역할 필터 - 단독)
│   ├── GroupFilter (소속 그룹 필터)
│   ├── GradeYearFilter (학년/학번 필터 - OR 관계)
│   └── AppliedFilters (적용된 필터 칩)
├── MemberListContent (목록 영역)
│   ├── [Desktop] MemberTable → MemberTableRow
│   └── [Mobile] MemberCardList → MemberCard
└── MemberListFooter (페이지네이션)
```

## 재사용 가능한 공통 컴포넌트

### 1. RoleBadge (역할 배지)
**파일**: `lib/presentation/widgets/member/role_badge.dart`

**용도**: 멤버의 역할을 시각적으로 표시

**Props**:
- `roleName: String` (필수)
- `backgroundColor: Color?` (옵션)
- `textColor: Color?` (옵션)

**사용 예시**: 멤버 카드, 멤버 테이블, 권한 관리 UI

### 2. MemberInfoRow (정보 행)
**파일**: `lib/presentation/widgets/member/member_info_row.dart`

**용도**: 아이콘 + 라벨 + 값 형태의 정보 행

**Props**:
- `icon: IconData` (필수)
- `label: String` (필수)
- `value: String` (필수)

**사용 예시**: 학번, 학년, 가입일 표시

### 3. MemberCard (멤버 카드)
**파일**: `lib/presentation/widgets/member/member_card.dart`

**용도**: 모바일 및 카드형 목록에서 멤버 표시

**Props**:
- `member: GroupMember` (필수)
- `trailing: Widget?` (옵션 - 체크박스, 드롭다운 등)
- `onTap: VoidCallback?` (옵션)

### 4. MemberTableRow (테이블 행)
**파일**: `lib/presentation/widgets/member/member_table_row.dart`

**용도**: 데스크톱 테이블 레이아웃에서 멤버 표시

**Props**:
- `member: GroupMember` (필수)
- `actions: List<Widget>?` (옵션 - 역할 드롭다운, 메뉴 등)

## 필터 패널 UI

### 전체 레이아웃

```
┌─────────────────────────────────────────┐
│ 멤버 필터                                │
├─────────────────────────────────────────┤
│ 역할 (선택 시 단독 필터) ⓘ               │
│ ☐ 그룹장  ☐ 교수  ☐ 멤버  ☐ ...         │
├─────────────────────────────────────────┤
│ 소속 그룹 (하위 그룹) ⓘ                  │
│ ☐ AI 학회                               │
│ ☐ 프로그래밍 동아리                      │
│                                         │
│ 학년 또는 학번 ⓘ                         │
│ ─────────────────────────────────────   │
│ 학년                                    │
│ ☐ 1학년  ☐ 2학년  ☐ 3학년  ☐ 4학년      │
│                                         │
│ 학번 (입학년도)                          │
│ [학번 선택 ▼]                           │
│   ☐ 24학번 (15명)                       │
│   ☐ 25학번 (8명)                        │
├─────────────────────────────────────────┤
│ 적용된 필터:                            │
│ [그룹: AI학회 ×] [학년: 2,3학년 ×]      │
│ [모두 지우기]                           │
├─────────────────────────────────────────┤
│ 결과: 15명                              │
└─────────────────────────────────────────┘
```

### 상호작용 규칙

**역할 필터 선택 시**:
- 모든 일반 필터 비활성화 (회색 처리)
- 이미 선택된 일반 필터 자동 초기화

**일반 필터 선택 시**:
- 역할 필터 비활성화 (회색 처리)
- 역할 필터 선택 해제

**학년/학번 그룹**:
- 시각적 그룹핑 (박스 또는 배경색)
- "또는" 라벨 명시적 표시

## 반응형 레이아웃

### 데스크톱 (1024px 이상)

```
┌─────────────────────────────────────────┐
│ 필터 패널 (좌측 300px)  │  멤버 목록     │
│ [역할 필터]             │  (테이블)      │
│ [소속 그룹 필터]        │                │
│ [학년/학번 필터]        │  ┌───────────┐ │
│ [적용된 필터]           │  │ 이름 역할  │ │
│ [결과: N명]            │  └───────────┘ │
└─────────────────────────────────────────┘
```

### 모바일 (768px 이하)

```
┌─────────────────────────┐
│ [필터 🔽] (접기/펼치기)  │
├─────────────────────────┤
│ 멤버 목록 (카드)         │
│ ┌─────────────────────┐ │
│ │ MemberCard          │ │
│ └─────────────────────┘ │
└─────────────────────────┘
```

**모바일 필터 UI**: 바텀 시트 또는 전체 화면 오버레이


## 디자인 토큰

### 색상
- 역할 필터 배경: `gray-50` (비활성), `blue-50` (활성)
- 구분선: `gray-200`
- 비활성 필터: `gray-400` (텍스트), `gray-100` (배경)
- 적용된 필터 칩: `blue-100` (배경), `blue-700` (텍스트)

### 간격
- 필터 섹션 간격: `16px`
- 체크박스 간격: `12px`
- 필터 칩 간격: `8px`

### 타이포그래피
- 필터 제목: `titleMedium` (16px, 600)
- 체크박스 라벨: `bodyMedium` (14px, 400)
