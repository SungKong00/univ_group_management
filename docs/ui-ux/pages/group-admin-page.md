# 그룹 관리 페이지 (Group Admin Page)

> **소유자**: `frontend-specialist`
> **상태**: `구현 중`

이 문서는 그룹 관리 페이지의 UI/UX 및 기능 명세를 정의합니다.

## 1. 페이지 개요

-   **경로**: `/group/{groupId}/admin` (개념적 경로, 실제 라우팅은 워크스페이스 내에서 처리)
-   **진입점**:
    -   워크스페이스 좌측 채널 목록 하단의 "그룹 관리" 버튼
-   **접근 권한**: `GROUP_MANAGE`, `MEMBER_MANAGE`, `CHANNEL_MANAGE`, `RECRUITMENT_MANAGE` 권한 중 하나 이상을 보유한 사용자

## 2. 핵심 기능 (Phase 1 - 현재)

-   **준비 중 상태 표시**:
    -   페이지에 진입하면 중앙에 아이콘과 함께 "그룹 관리 (준비 중)" 텍스트를 표시합니다.
    -   이는 현재 기능이 개발 중임을 사용자에게 명확히 알리는 역할을 합니다.

## 3. UI 구성 요소 (Phase 1 - 현재)

-   **중앙 컨텐츠**:
    -   `Icon(Icons.settings_outlined)`: 설정 아이콘
    -   `Text('그룹 관리')`: 페이지 제목 (스타일: `AppTheme.displaySmall`)
    -   `Text('그룹 관리 페이지 (준비 중)')`: 설명 텍스트 (스타일: `AppTheme.bodyLarge`)

## 4. 향후 확장 계획 (Phase 2 이후)

-   **그룹 정보 관리**: 그룹 이름, 설명, 공개 범위 등 수정
-   **멤버 관리**: 멤버 목록 조회, 역할 변경, 강제 탈퇴
-   **가입 관리**: 가입 신청 목록 확인 및 승인/거절
-   **채널 관리**: 채널 목록, 생성, 수정, 삭제
-   **모집 관리**: 모집 공고 관리
-   **권한 관리**: 커스텀 역할 생성 및 권한 할당

## 5. 상호작용

-   사용자가 좌측 네비게이션에서 "그룹 관리" 버튼을 클릭하면, 메인 컨텐츠 영역이 이 페이지로 전환됩니다.
