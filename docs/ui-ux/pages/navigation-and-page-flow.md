# 네비게이션 및 페이지 플로우 (Navigation and Page Flow)

애플리케이션의 핵심 네비게이션 구조와 페이지 전환 UI/UX 정책을 정의합니다.

## 개요

기본 레이아웃, 상단/좌측 네비게이션, 페이지 전환, 뒤로가기 동작을 다룹니다. 워크스페이스 특수 플로우는 별도 문서 참조.

## 기본 레이아웃

- **상단 네비게이션**: 뒤로가기 + 페이지 정보 (브레드크럼)
- **좌측 네비게이션**: 주요 메뉴 (홈, 워크스페이스, 캘린더, 나의 활동, 프로필)

## 상단 네비게이션

### 뒤로가기 버튼
- 위치: 좌측
- 기능: 실제 이동 경로 기반 이전 페이지

### 페이지 정보 (브레드크럼)
- 위치: 뒤로가기 우측
- **주제목**: 20px/600/neutral900
- **경로**: 14px/400/neutral600, ">" 구분
- 예: "워크스페이스 > 컴퓨터공학과 > 공지사항"

## 좌측 네비게이션

### 메뉴 구성
- 아이콘 + 페이지 이름 + 설명
- 선택 표시: 우측 `|` 표시

## 페이지 전환

### 일반 전환
- 콘텐츠 영역만 새로 로드
- 상단/좌측 네비게이션 유지
- 브레드크럼만 동적 변경

### 상태 유지
- 스크롤 위치, 입력 데이터 유지
- 중단 지점에서 작업 재개 가능

### 뒤로가기
- 실제 이동 경로 기억
- 좌측 네비게이션 선택 상태 동기화
- 예: 홈 → 워크스페이스 → (뒤로) → 홈

## 워크스페이스 특수 플로우

워크스페이스는 독자적 네비게이션 플로우를 가집니다:

**문서**: [workspace-navigation-flow.md](workspace-navigation-flow.md)

특수 동작:
- 좌측 네비게이션 축소
- 채널 네비게이션 바 표시
- 게시글/댓글 시스템
- 모바일/반응형
