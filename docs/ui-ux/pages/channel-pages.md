# UI/UX 명세서: 채널 페이지

본 문서는 워크스페이스의 핵심 소통 공간인 채널의 생성, 탐색, 상호작용에 대한 상세한 UI/UX를 정의합니다.

## 1. 기본 생성 채널

워크스페이스가 처음 생성될 때, 시스템은 두 개의 기본 채널을 자동으로 생성합니다.

-   **자유 채널**: 모든 멤버가 자유롭게 글을 읽고 쓸 수 있는 권한을 가집니다.
-   **공지 채널**: 관리자만 글을 쓸 수 있고, 다른 모든 멤버는 읽기만 가능한 권한을 가집니다.

이 채널들의 권한 또한 생성 후 '채널 설정'에서 언제든지 변경할 수 있습니다.

## 2. 사용자 채널 생성 플로우

워크스페이스 생성 후, 사용자는 필요에 따라 채널을 직접 추가할 수 있습니다.

### 2.1. 생성 시작

-   채널 목록 상단 또는 하단의 `+ 채널 추가` 버튼을 클릭하여 새 채널 생성을 시작합니다.

### 2.2. 생성 모달

버튼 클릭 시, 최소한의 정보만 입력하는 간단한 모달(Modal) 팝업이 나타납니다.

1.  **채널 이름**: 필수 입력 필드입니다.
2.  **채널 설명**: 채널의 목적을 설명하는 선택적 입력 필드입니다.
3.  **버튼**:
    -   **[생성하기]**: 입력한 이름으로 채널을 생성합니다.
    -   **[취소]**: 모달을 닫습니다.

**※ 중요**: 채널 생성 시점에는 권한을 설정하는 단계가 없습니다.

### 2.3. 생성 후 권한 설정

-   채널이 생성되면, 기본적으로는 생성자를 제외한 다른 멤버들에게는 보이지 않거나 아무 권한이 없는 상태입니다.
-   채널 생성자는 즉시 **'채널 설정 > 권한'** 탭으로 이동하여, 그룹 내 역할(예: 정회원, 신입회원)별로 '채널 보기', '글쓰기' 등의 권한을 직접 부여해야 합니다.

## 3. 채널 메인 뷰

사용자가 채널 목록에서 특정 채널을 선택했을 때 표시되는 메인 콘텐츠 영역입니다.

### 3.1. 헤더

-   **채널 정보**: 좌측에 `#` 아이콘과 함께 채널 이름, 그리고 그 아래에 채널 설명을 표시합니다.
-   **설정 버튼**: 우측에 `⚙️` (톱니바퀴) 아이콘을 배치하여 '채널 설정' 메뉴로 진입할 수 있도록 합니다. 이 버튼은 채널 관리 권한이 있는 사용자에게만 보입니다.

### 3.2. 콘텐츠 (게시글 목록)

-   `workspace-pages.md`에 정의된 대로, 시간순으로 정렬된 메시지 스트림을 표시합니다.

### 3.3. 메시지 입력창

-   페이지 하단에 위치하며, 현재 채널에서 사용자에게 부여된 권한에 따라 상태가 결정됩니다.
-   **쓰기 권한이 있는 경우**: 입력창이 활성화됩니다.
-   **쓰기 권한이 없는 경우**: 입력창이 비활성화되며, "이 채널은 메시지를 작성할 권한이 없습니다."와 같은 안내 문구가 표시됩니다.

## 4. 채널 설정 페이지

채널의 모든 관리는 이곳에서 이루어집니다.

### 4.1. 탭 구조

1.  **일반 (General)**: 채널 이름과 설명을 수정합니다.
2.  **권한 (Permissions)**: 채널의 핵심 기능입니다.
    -   **UI**: 권한의 종류를 행(Row)으로, 해당 권한을 부여할 역할을 설정하는 영역을 열(Column)으로 하는 리스트 형태의 UI를 제공합니다.
    -   **상호작용**: 각 권한 항목(예: '게시글 쓰기') 옆에, 해당 권한을 가질 역할(예: '운영진', '정회원')들을 추가하거나 제거할 수 있는 태그(Tag) 또는 드롭다운 형태의 입력 필드를 제공합니다.
        -   | 권한 종류 | 부여된 역할 |
            | :--- | :--- |
            | 채널 보기 | `[운영진] [정회원] [신입회원] [ + ]` |
            | 게시글 쓰기 | `[운영진] [ + ]` |
            | 댓글 쓰기 | `[운영진] [정회원] [ + ]` |
    -   **비공개 채널 구현**: '채널 보기' 권한에서 특정 역할(예: '정회원')을 제거하면, 해당 역할의 사용자에게는 채널이 보이지 않게 되어 실질적인 '비공개' 상태가 됩니다.
3.  **참여자 (Members)**: 현재 채널의 '채널 보기' 권한이 있는 모든 멤버 목록을 보여줍니다.
4.  **관리 (Danger Zone)**: 채널 보관(Archive) 또는 영구 삭제를 수행합니다.
