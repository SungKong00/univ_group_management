# 모집 관리 페이지 (Recruitment Admin Pages)

모집 담당자가 사용하는 모집 공고 작성/수정 및 지원자 관리 페이지입니다.

## 개요

`RECRUITMENT_MANAGE` 권한 사용자는 모집 공고를 작성/수정하고, 지원자를 심사합니다.

## 접근 경로

워크스페이스 > 관리자 메뉴 > 모집 관리

## 페이지 구조

탭 구조: [모집 공고 관리] [지원자 관리]

**구현**: `TabController` 사용, 반응형 (768px 기준)

## 1. 모집 공고 작성/수정

### 질문 입력 UI
지원서 질문 설정 (최대 5개, 각 1,000자 이하)

**동작**:
- 초기: 질문 1개
- "➕ 질문 추가" → 새 필드 추가 (최대 5개)
- [X 삭제] → 개별 삭제 (최소 1개 유지)
- 5개 도달 시 추가 버튼 비활성화

**반응형**:
- 데스크톱: 전체 너비, 삭제 버튼 우측
- 모바일: 축소, 아이콘만 표시

## 2. 지원자 관리

### 구성 요소
- **상태 필터**: 전체, 신청 완료, 승인, 반려, 철회
- **지원자 리스트**: 프로필, 닉네임, 지원일, 상태 (Chip)
- **심사 액션**: 승인/반려 버튼 ("신청 완료" 상태만)
- **반려 Dialog**: 사유 입력

### 정렬 규칙
- 철회된 지원서: 하단에 비활성화 표시
- 재지원: 활성/철회 지원서 모두 각각 표시

### 반응형

#### 데스크톱 (768px+)
테이블: [지원자 | 지원일 | 상태 | 심사자 | 액션]
- `Expanded(flex)` 비율
- 우측 승인/반려 버튼

#### 모바일
카드 레이아웃: 아바타 + 정보 + 버튼
- 패딩 16px, 간격 12px
