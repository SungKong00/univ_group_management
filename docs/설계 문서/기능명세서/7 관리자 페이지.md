**1. 관리 홈 및 접근 정책**

- **접근 방식:** 하나 이상의 관리 권한을 가진 사용자에게만 '관리' 버튼이 노출되며, 이를 통해 '관리 홈'으로 진입한다.
- **UI 컨셉 (Toss UI/UX 적용):**
    - 단순 텍스트 목록이 아닌, **아이콘과 설명이 포함된 카드/리스트 형태**의 메뉴를 제공한다.
    - **맥락에 맞는 정보**를 함께 표시하여 관리자가 다음에 할 일을 바로 인지하게 한다. (예: `멤버 관리` 메뉴 옆에 `가입 대기 N명` 뱃지 표시)
    - 사용자에게 **권한이 있는 메뉴만 목록에 표시**한다.

**2. MVP 기능 목록**

- `멤버 관리` (가입 대기 인원 표시)
- `역할 관리`
- `채널 관리`
- `지도교수 관리` (그룹장에게만 노출)
- `그룹 정보 수정`
- `그룹 삭제`

**3. 핵심 기능별 상세 흐름 (Toss UI/UX 적용)**

- **3.1. 역할 및 채널 관리**
    - **흐름 변경:** 여러 단계의 대화형 흐름을 폐기하고, **'한 화면 설정'** 방식을 도입한다.
    - **(예시)** '역할 생성' 클릭 시, **[역할 이름 입력란]**과 **[권한 목록 토글 스위치]**가 한 페이지에 모두 나타난다. 사용자는 한눈에 보며 설정하고 하단의 '저장' 버튼을 누르면 즉시 완료된다.
- **3.2. 그룹 삭제**
    - **흐름 변경:** 삭제 시 **바텀 시트(Bottom Sheet)**를 통해 삭제될 데이터 정보를 명확히 안내하고, 사용자가 **그룹 이름을 직접 입력**해야만 활성화되는 **빨간색 [삭제] 버튼**을 눌러 최종 확인하는 강력한 확인 절차를 거친다.

### 5. API 명세 (UI/UX 명세서 v1.0 및 통합 표준 기준)

| 기능 | HTTP Method | URI | Request Body | Response Body | 설명 |
| --- | --- | --- | --- | --- | --- |
| **관리 홈 요약 조회** | **GET** | **/groups/{groupId}/admin/stats** | | `AdminStatsDto` | 관리 홈에 필요한 통계(가입 대기 인원, 멤버 수 등)를 조회합니다. |
| **멤버 관리 API** | (다수) | /groups/{groupId}/members... | | | '권한/멤버 관리' 명세서에 정의된 RESTful API를 재사용합니다. |
| **역할 관리 API** | (다수) | /groups/{groupId}/roles... | | | '권한/멤버 관리' 명세서에 정의된 RESTful API를 재사용합니다. |
| **채널 관리 API** | (다수) | /groups/{groupId}/channels... | | | '그룹 워크스페이스' 명세서에 정의된 RESTful API를 재사용합니다. |
| **지도교수 목록 조회** | **GET** | **/groups/{groupId}/advisors** | | `List<UserDto>` | 현재 그룹의 지도교수 목록을 조회합니다. |
| **지도교수 추가** | **POST** | **/groups/{groupId}/advisors** | `AddAdvisorRequest` | (204 No Content) | 이메일로 지도교수를 검색하여 추가합니다. |
| **지도교수 삭제** | **DELETE** | **/groups/{groupId}/advisors/{userId}** | | (204 No Content) | 지도교수를 그룹에서 제외합니다. |
| **그룹 정보 수정** | **PATCH** | **/groups/{groupId}** | `UpdateGroupRequest` | `GroupDto` | 그룹의 이름, 설명 등 기본 정보를 수정합니다. |
| **그룹 삭제** | **DELETE** | **/groups/{groupId}** | `DeleteGroupRequest` | (204 No Content) | 그룹 및 모든 데이터를 영구 삭제합니다. (이름 재확인 포함) |

### 6. DTO (Data Transfer Object) 정의

- **AdminStatsDto**
    - `pendingMemberCount: Int`
    - `totalMemberCount: Int`
    - `roleCount: Int`
    - `channelCount: Int`
- **UpdateGroupRequest**
    - `name: String`
    - `description: String`
    - `tags: List<String>`
- **AddAdvisorRequest**
    - `email: String`
- **DeleteGroupRequest**
    - `confirmGroupName: String`
- **UserDto**
    - `userId: Long`
    - `nickname: String`
    - `email: String`
    - `profileImageUrl: String`
