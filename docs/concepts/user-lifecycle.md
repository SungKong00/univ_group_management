# 사용자 여정 (User Lifecycle)

가입부터 활동까지 사용자가 경험하는 전체 과정입니다.

## 전체 흐름

```
Google 로그인 → 프로필 작성 → 학과 자동 가입 → 그룹 활동 시작
                                           ↓ (선택)
                                    교수 신청 → 승인 → 교수 권한 획득
```

## 사용자 상태별 경험

### 1. 처음 방문 (Google 로그인 후)
- **기본 정보**: Google에서 이메일, 이름 자동 가져옴
- **접근 제한**: 프로필 완성 전까지 제한된 접근
- **다음 단계**: 프로필 작성으로 자동 안내

### 2. 프로필 작성 중
필수 정보:
- **닉네임**: 플랫폼 내 별명
- **이름**: 실명
- **학과**: 목록에서 선택 (미선택 시 "AI/SW 학부" 자동 배정)
- **학번**: 학생 번호

**특징**: 완성 후 바로 모든 기능 이용 가능

### 3. 일반 학생
- **소속**: 선택한 학과 그룹에 자동 가입
- **기본 활동**: 그룹 가입, 워크스페이스 활동, 일반 권한
- **가능한 일**:
  - 그룹 탐색 및 가입
  - 워크스페이스 채널 접근
  - 게시글 작성/댓글 (권한 범위 내)
  - 비공식 일정 생성

### 4. 교수 승인 대기
- **신청 조건**: 교수임을 증명하는 사유 작성
- **임시 권한**: 승인 전까지는 학생 권한
- **상태 표시**: 프로필에 "교수 승인 대기 중" 배너
- **결과**: 알림으로 승인/거부 확인

### 5. 승인된 교수
- **관리 권한**: 소속 학과 내 그룹들에 대한 관리 권한
- **교수 기능**:
  - 학생 그룹 생성 승인/거부
  - 감독/지도 기능
  - 캘린더 관리, 공식 일정 생성
- **높은 신뢰도**: 시스템 내 높은 권한과 책임

## 자동 학과 가입

### 학과 자동 배정
프로필 완성 시:
1. 선택한 학과 그룹에 **자동으로 가입**
2. 그룹이 없으면 자동으로 생성

예: "AI/SW 학부" 선택 시
- 한신대학교 (최상위) 확인/생성
- AI/SW 대학 (중간) 자동 생성
- AI/SW 학부 (최하위) 자동 생성
- 사용자 자동 가입

### 미선택 시
"AI/SW 학부"로 **자동 배정** (나중에 변경 가능)

## 교수 신청 과정

### 신청하기
1. 프로필에서 "교수 신청" 버튼 클릭
2. 신청 이유 작성 (교수 자격 설명)
3. 대기 상태로 전환

### 관리자 검토
1. 신청서 확인
2. 적절성 판단 후 승인/거부 결정
3. 신청자에게 알림 발송

### 승인 후
- 교수 권한 즉시 활성화
- "교수 계정이 승인되었습니다" 알림 수신
- 그룹 관리 기능 즉시 사용 가능

## 계정 관리

### 일시 정지 (휴학/휴가)
- **정지 사유**: 휴학, 교환학생 등 기록
- **복귀 예정일**: 언제 돌아올지 설정
- **데이터 보관**: 모든 정보 그대로 보관
- **복귀**: 정지 해제하면 즉시 복구

### 계정 삭제
영구 삭제 전 자동 처리:
1. **오너 권한 이양**: 소유 그룹의 오너권을 다른 멤버에게 이양
2. **그룹 탈퇴**: 모든 그룹에서 자동 탈퇴
3. **컨텐츠 익명화**: 게시글/댓글은 "탈퇴한 사용자"로 표시
4. **개인정보 삭제**: 완전 삭제

## 실제 사용 흐름 예시

```
1단계: 박민수가 Google로 로그인
     → 시스템에 신규 사용자 생성

2단계: 프로필 작성
     - 닉네임: minsoo
     - 학과: AI/SW 학부 선택
     - 학번: 2023001
     ✓ 프로필 완성

3단계: 자동 처리
     - "AI/SW 학부" 그룹에 자동 가입
     - 워크스페이스 접근 가능
     - 기본 2채널 (공지사항, 자유게시판) 보임

4단계: 플랫폼 활동
     - 다른 그룹 탐색 및 가입
     - 채널에서 게시글 작성/댓글
     - 그룹 캘린더 참여

5단계 (선택): 교수 신청
     - "교수 신청" 버튼 클릭
     - 교수 자격 설명 작성
     - 관리자 검토 대기

6단계 (선택): 교수 승인
     - 관리자 승인 완료
     - 교수 권한 활성화
     - 그룹 관리 기능 사용 가능
```
