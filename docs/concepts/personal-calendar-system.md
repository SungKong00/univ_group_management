# 개인 캘린더 시스템 (Personal Calendar System)

사용자 개인의 시간표와 일정을 관리하는 시스템입니다.

## 개요

**시간표**와 **캘린더** 두 가지로 구성됩니다.
- **시간표**: 고정된 주간 일정 (강의, 정기 회의, 아르바이트)
- **캘린더**: 모든 일정을 통합 관리하는 월/주/일 뷰

## 시간표 (Timetable)

### 특징
- **뷰**: 주간(Weekly) 뷰만 제공 (고정 스케줄 직관적 표시)
- **목적**: 다른 기능(최적 시간 추천)의 기반 데이터
- **구성**:
  - 학교 강의 (관리자 등록 후 사용자가 선택)
  - 개인 반복 일정 (요일, 시간, 제목 직접 입력)

### 사용 예시

```
월 09:00~11:00  데이터베이스
    14:00~16:00  웹개발
화 10:00~12:00  알고리즘
    14:00~16:00  알고리즘 실습
...
```

시간표에 등록된 시간은 **그룹 일정 생성 시 최적 시간 추천 분석에 포함**되어,
개인의 참석 불가능한 시간을 미리 제외합니다.

## 캘린더 (Calendar)

### 특징
- **뷰**: 월(Month), 주(Week), 일(Day) 뷰 제공
- **목적**: 개인이 확정한 모든 일정을 한곳에서 관리
- **관리 대상**:
  - **개인 이벤트**: 병원 예약, 친구와의 약속, 여행 계획 등
  - **참여 그룹 일정**: 그룹 캘린더에서 '참여' 의사를 밝힌 모든 일정

### 동기화 방식

그룹 캘린더에서 초대받은 일정에 '참여(ACCEPTED)'를 누르면:
1. **개인 캘린더에 자동 반영** → 사용자는 자신의 모든 확정 일정을 한곳에서 확인
2. 그룹 캘린더에서 '불참(DECLINED)' 처리 시 개인 캘린더에서도 제거

### 실제 사용 흐름

```
1. 그룹 캘린더에서 "11월 20일 팀 미팅" 초대 받음
2. "참여합니다" 버튼 클릭
3. ↓ 자동 동기화
4. 개인 캘린더 11월 20일에 "팀 미팅" 표시됨
5. 사용자는 개인 캘린더에서 모든 약속 확인 가능
```

## 권한

- **조회**: 본인의 시간표/캘린더만 조회 가능
- **생성/수정/삭제**: 본인 만 관리
