# 모집 시스템 (Recruitment System)

그룹이 신규 멤버를 모집하고 관리하는 시스템입니다.

## 개요

그룹은 **모집 게시글**을 작성하여 신규 멤버를 모집하고, 지원자들의 **지원서**를 심사하여 가입을 승인합니다.

## 핵심 규칙

### 모집 게시글
- **그룹당 1개**: 동시에 활성 모집은 1개만 가능
- **마감일**: 선택사항 (설정 후 자동 마감)
- **조기 마감**: 권한자가 마감일 전 마감 가능
- **지원자 수**: 공개 여부 선택 가능
- **수정 정책**: 기존 지원자에 영향 없음
- **삭제 정책**: 지원자 있으면 soft-delete (논리적 삭제)

### 지원서
- **중복 지원 방지**: 1개 모집당 1회만 (철회 후 재지원 가능)
- **상태**: 신청 완료 → 승인/반려 또는 철회
- **지원자 철회**: 심사 완료 전에만 가능
- **자동 가입**: 승인 시 그룹에 자동으로 가입
- **반려 사유**: 반려 시 심사 의견 확인 가능

### 아카이브
마감된 모집은 자동으로 아카이브로 이동 (권한자만 조회 가능)

## 권한 체계

| 기능 | 필요 권한 | 추가 조건 |
|------|---------|---------|
| 모집 생성/수정/삭제 | `RECRUITMENT_MANAGE` | 그룹 멤버여야 함 |
| 조기 마감 | `RECRUITMENT_MANAGE` | - |
| 지원서 심사 | `RECRUITMENT_MANAGE` | 해당 그룹 소속 |
| 모집 공고 조회 | (없음) | 모든 사용자 |
| 지원서 제출 | (없음) | 미가입자만 가능 |
| 본인 지원서 확인 | (없음) | 본인만 |

## 실제 사용 흐름

```
1. 그룹 운영진
   - "2025 1학기 신입 기수 모집" 게시글 작성
   - 마감일: 2025-11-15
   - 최대 지원자: 30명

2. 사용자
   - 공개 모집 목록에서 확인
   - 지원서 제출 (동기, 질문 답변)
   - 상태: "신청 완료"

3. 운영진
   - 지원서 목록 확인
   - 심사하여 승인 또는 반려
   - 승인 시 지원자 자동 가입

4. 지원자
   - 마감일 도달 시 모집 자동 종료
   - 아카이브에서 지원 이력 확인
```

## 지원서 상태 전이

```
[신청 완료]
  ├─ → [승인] → 그룹 자동 가입
  ├─ → [반려] → 심사 의견 확인 가능
  └─ → [철회] (신청자가 심사 전 철회 가능)
```
